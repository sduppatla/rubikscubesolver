This program is a rubik's cube solver that can take in a rubik's cube and return a solution.

How to run:
To run this project make sure to download all of the files and then run the file called RubiksCubeTKinter.

The necessary libraries for this project are:
    TKinter
    PIL

Shortcut Commands
    's' for randomly generated scramble
    'Space' can be held down to solve the cube rapidly
